# Модули операций с тензорами
Данный модуль включает или будет включать в себя операции и реализацию трейтов, направленные на следующие операции:

- [X] [Сложение](./add.rs)
- [X] [Вычитание](./sub.rs)
- [] Умножение
    - [] Простое перемножение с перебором
    - [] Перемножение с предварительным (физическим) транспонированием
    - [] Перемножение в несколько потоков
    - [] Перемножение с предварительным транспонированием по чанкам
    - [] Умножение тензора на скаляр
- [] Деление (с нюансом)
    - [] Умножение на обратную матрицу
    - [] Поэлементное деление (Hadamard Division)
    - [] Деление на скаляр
    - [] Обратная матрица (Matrix Inverse)
    - [] СЛАУ (Matrix Solve)
